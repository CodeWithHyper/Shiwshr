# Swishr 🏀

A modern basketball scoreboard web app with a clean courtside UI, shot clock controls, foul tracking, bonus indicators, quick edits, fullscreen mode, and saved game state.

## About
Swishr is built for fast in-game operation on desktop and mobile. It focuses on quick controls, clear visibility, and practical scoring-table features.

## Features
- Home/Away score tracking (`+1`, `+2`, `+3`, `-1`)
- Game clock with start/pause/resume
- Shot clock controls (`24 SEC`, `14 SEC`) with visual ring
- Period tracking (`Q1` to `Q4`, then `OT`) and game-over flow
- Team foul counters with `BONUS` and `DOUBLE BONUS` indicators
- Editable team names
- Quick Edit modal for period/time/shot clock correction
- Fullscreen mode for courtside/table display
- Local storage persistence (refresh-safe state)
- Undo last action
- Clear saved game action
- Responsive layout for mobile portrait/landscape

## Tech Stack
- HTML
- CSS
- JavaScript
